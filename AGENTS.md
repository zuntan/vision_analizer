# 開発内容

- 画像をドラッグすると、その画像を解析してテキスト化する web アプリケーションを作成する

## 使用言語、ライブラリ等
- python + venv
  - Uvicorn + FastAPI
  - htmx
  - logging
  - httpx 
  - Pillow
  - pydantic
- tailwind.css 
- Lucide Icons

## 画面と動作

- タイトル vision_analizer
- `ドラッグエリア`
  - ここにファイルをドラッグする
  - ファイルは画像ファイルのみとする。JPEG (.jpg, .jpeg), PNG, BMP, WEBP
  - ファイルがドラッグされたら、アップロード。そのファイル内容を /tmp にテンポラリファイル名で保存し、そのファイル名を記憶して、プレビュー欄に表示
- `解析エリア`
  - `プロンプト入力欄`
    - ドロップダウンでプロンプト候補を指定可能とする。デフォルトプロンプト文章は、ドロップダウンの先頭値とする
  - ドラッグされたファイルのプレビュー
    - 初期状態、クリア状態では、空欄
    - ファイルがアップロードされていたら、その画像ファイルを表示
  - 解析されたテキストを表示するエリア
  - コピーボタン
    - 解析されたテキストをクリップボードにコピー
  - 再解析ボタン
  - クリアボタン

## 設定ファイル

toml 形式

- openaiアクセスポイント
  - 初期値: http://kuro.local:8081/v1
- システムプロンプト文言
  - 初期値: 「日本語で応答せよ」
- プロンプト文言集
  - 「画像を簡素に説明せよ」
  - 「画像をできるだけ詳しく説明せよ」
  - 「ＵＩ形状を優先して詳しく説明せよ」
  - 「テキストを正確に文字化せよ」
